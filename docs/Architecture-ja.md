# AEVP アーキテクチャ

AEVP は、省エネルギーに関する時系列データを用いて、
基準消費量を予測し、実績値との差分から省エネルギー貢献量を算出し、
それをポイント、スコア、または評価値に変換する小型プロトタイプである。

## 全体の流れ

1. 時系列データを入力する。
2. 予測部で基準消費量を推定する。
3. 基準消費量と実績消費量を比較する。
4. 差分を省エネルギー貢献量として計算する。
5. 省エネルギー貢献量をポイントまたは評価値に変換する。

## 主要コンポーネント

| コンポーネント | 役割 |
| --- | --- |
| Predictor | 予測器の共通インターフェース |
| MovingAveragePredictor | 移動平均による軽量予測 |
| ExponentialSmoothingPredictor | 指数平滑による軽量予測 |
| EnergySavingScore | 省エネルギー貢献量を点数化する |
| CappedEnergySavingScore | 点数に上限・下限を設ける |
| AEVPPipeline | 予測、比較、点数化を統合する |
| AEVPResult | 処理結果を保持する |
