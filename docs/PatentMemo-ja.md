# AEVP 特許メモ

この文書は、AEVP の実装を特許出願の検討に接続するためのメモである。
正式な特許明細書ではなく、発明の核を整理するための作業文書である。

## 発明名候補

省エネルギー貢献量に基づく価値ポイント算出システム、方法、およびプログラム

## 背景

設備、建物、店舗、工場、住宅、または小型拠点において、
消費電力や運転状態の時系列データを用いて省エネルギー効果を評価する需要がある。

しかし、単に実際の消費量を見るだけでは、
「本来どれくらい消費していたはずか」という基準が曖昧になる。

そこで、時系列予測により基準消費量を推定し、
実績値との差分を省エネルギー貢献量として評価する。

## 解決手段

本システムは、過去のエネルギー使用量または設備状態の時系列データから、
将来または対象期間の基準消費量を予測する。

次に、予測された基準消費量と実際の消費量とを比較し、
実際の消費量が基準消費量より低い場合、その差分を省エネルギー貢献量として算出する。

さらに、省エネルギー貢献量をポイント、スコア、報酬値、評価値、または決済補助値に変換する。

## 構成要素

- 時系列データ取得部
- 予測部
- 基準消費量算出部
- 実績消費量取得部
- 省エネルギー貢献量算出部
- ポイント変換部
- 上限・下限補正部
- 小型計算機またはエッジ装置上の実行部

## ソフトウェア対応

| 特許上の機能 | 現在の実装 |
| --- | --- |
| 予測部 | MovingAveragePredictor, ExponentialSmoothingPredictor |
| 省エネルギー貢献量算出部 | EnergySavingScore |
| 上限・下限補正部 | CappedEnergySavingScore |
| 統合処理部 | AEVPPipeline |
| 結果出力 | AEVPResult |

## 請求項の核の候補

1. 時系列データに基づいて基準消費量を予測する予測部を備えること。
2. 予測された基準消費量と実績消費量との差分に基づいて省エネルギー貢献量を算出すること。
3. 省エネルギー貢献量をポイントまたは評価値に変換すること。
4. ポイントまたは評価値に上限、下限、係数、時間帯補正、または利用者区分補正を適用すること。
5. 小型 PC、エッジ装置、または組込み装置上で実行可能であること。
